En esta directorio se encuentran tres archivos y dos directorios:
	* generate_dataset.py: Se encarga de hacer requests al API de Echonest para generar
	  el data set (Se requiere tener instalada la librería Pyechonest).
	* filter_dataset.py: Se encarga de balancear el data set obtenido.
	* Informe - Clasificador de generos musicales.pdf: El informe con los resultados.
	* data_set:
		* songs_data_set.tab: El data set obtenido, con el formato utilizado en Orange.
		* balanced_songs_data_set.tab: el data set balanceado, con el formato utilizado en
		  orange.
	* graphics: Contiene las imagenes y los resultados obtenidos usando las herramientas de
	  Orange.
